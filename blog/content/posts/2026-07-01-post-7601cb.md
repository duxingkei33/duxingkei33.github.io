## 🔧 刷机指南

> 原厂固件使用私有加密协议，**只能配合汉朔商用基站**。刷第三方固件后才能通过 BLE 自由控制。

### 前置确认

拆电池后盖看主板 — 主控是 **TLSR8359** 即可刷。Stellar-xx**N**x 系列（含 XL3N）通常都是 TLSR8359。

### 接线烧录

所需工具：USB-TTL 适配器（CH340/CP2102，**3.3V**）+ 电烙铁 + 浏览器（Chrome/Edge）

```
价签 PCB ←→ USB-TTL
VCC (3.3V) → 3.3V
GND        → GND
TX         → RX
RX         → TX
```

1. 焊四根线（GND / VCC / TX / RX），部分型号需焊 RTS 到 CH340 第三脚（或手动碰 GND）
2. 打开 [ATC WebSerial 刷机工具](https://atc1441.github.io/ATC_TLSR_Paper_UART_Flasher.html)
3. 选 .bin 固件，波特率 460800，点 Unlock → Write to Flash
4. 成功后屏幕自动刷新

### 编译固件

```bash
git clone https://github.com/atc1441/ATC_TLSR_Paper.git
cd ATC_TLSR_Paper/Firmware
# Linux
make -j4
# Windows
makeit.exe -j12
```
