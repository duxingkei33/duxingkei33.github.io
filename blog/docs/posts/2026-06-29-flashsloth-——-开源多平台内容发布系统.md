## 关于 FlashSloth

**FlashSloth（🦥⚡）** —— 树懒的速度，闪电的发布。

这是一个开源的多平台内容发布系统，写一次 Markdown 文章，即可一键发布到多个平台：

- WordPress
- 微信公众号
- 掘金
- 知乎
- CSDN
- RSS 订阅源

### 为什么要做 FlashSloth？

作为一名内容创作者，常常需要在多个平台同步发布文章，重复操作费时费力。FlashSloth 的目标就是解决这个问题——**写一次，发遍全网**。

### 技术特点

- 基于 Flask 构建，轻量易部署
- Python 3 纯代码，依赖极少
- 首次启动自动生成随机管理员账号，开箱即用
- 插件化 Publisher 架构，方便扩展新平台
- Web 管理后台，操作直观
